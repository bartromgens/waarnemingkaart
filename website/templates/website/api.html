Waarnemingen API

Waarnemingen data

waarnemingkaart.nl bevat een deel van de waarnemingen data van waarneming.nl.

Deze data is beschikbaar via de waarnemingkaart-api.

De kaart data is nog niet beschikbaar via deze API. Neem contact met ons op als je daar naar zoekt.

Verkennen

De data in de API is in de browser te verkennen via waarnemingkaart-api. Klik op de links om door de data te navigeren.

Voorbeeld

Een lijst van waarnemingen in json formaat is te vinden op /api/v1/observations/?format=json. Dit geeft de eerste 100 waarnemingen. De 'next' url is een link naar de volgende lijst (pagina) met 100 waarnemingen.

Voorwaarden

Deze data is beschikbaar onder de voorwaarden van waarneming.nl. Deze voorwaarden zijn hier te lezen. De data is volgens deze voorwaarden vrij voor niet-zakelijk gebruik door particulieren.

Waarschuwing

De API is in alpha staat en kan dus fouten bevatten en veranderen.
